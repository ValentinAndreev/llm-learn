LLM Learning System — проект для обучения с помощью AI. Система строит структурированные конспекты по запросу, позволяет их редактировать и использовать как основу для дальнейшего обучения. Конспект может работать и как ограничение объема материала, и как направление изучения: в его рамках система либо остается строго в заданных границах, либо при необходимости дополняет тему информацией из интернета.

На основе конспекта и связанных с ним материалов система генерирует карточки, тесты, краткие выжимки и вопросы на понимание, а также поддерживает диалоговое обучение и режимы проверки знаний, включая метод Фейнмана и экзаменатор. Архитектурно это набор независимых LLM-сценариев вокруг общего ядра знаний, с отдельным слоем интеграции для работы с разными API и локальными моделями.

Проект будет реализован на Rails 8.1, Ruby 4.0.2
